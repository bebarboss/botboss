A Python project for automatic ticket clicking using Selenium with a PyQt6 GUI.  
Designed for easy use without running commands in the terminal.

---

## Features
- Automatic ticket clicking based on scheduled time  
- User-friendly GUI built with PyQt6  
- Start/Stop the bot directly from the interface  
- Supports multiple websites/buttons (configurable)
